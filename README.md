# Chronic Kidney Disease Prediction App

An interactive machine learning web application that predicts Chronic Kidney Disease (CKD) using clinical features.

---

## Project Overview

This project demonstrates an end-to-end machine learning workflow:

- Data preprocessing
- Model training
- Model evaluation
- Performance comparison
- Feature importance analysis
- Interactive prediction interface
- Downloadable prediction report

---

## Models Used

1. Logistic Regression
2. Decision Tree Classifier

---

## Features

- Model comparison chart
- ROC Curve visualization
- Feature importance ranking
- Downloadable predictions CSV
- Interactive patient input form

---

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

---

## How to Run Locally

1. Clone the repository:

git clone https://github.com/bamideleadedeji/ckd-streamlit-app.git

2. Navigate into the folder:

cd ckd-streamlit-app


3. Install dependencies:

pip install -r requirements.txt

4. Run the app:

streamlit run streamlit_app.py


---

## Deployment

Deployed on Streamlit Cloud using:
- Branch: master
- File: streamlit_app.py
